# sagarsys.net

Personal portfolio website showcasing my work as a Senior JavaScript/TypeScript
Developer.

![Portfolio Screenshot](public/sagarsys.net%20screenshot.png)

## Tech Stack

-   **Framework**: Next.js 15 (App Router)
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS 4.0
-   **Animations**: Framer Motion
-   **UI Components**: Radix UI
-   **Content**: Markdown + GitHub API integration

## Getting Started

### Prerequisites

-   Node.js 22+ (or use the version specified in `.nvmrc` if available)

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build

```bash
npm run build
```

Generates optimized production build in the `out/` directory.
