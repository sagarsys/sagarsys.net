## Overview

Rework of a real-estate website for Kaufman & Broad, a publicly traded real estate development and construction company headquartered in France. Users can search for properties, learn about new opportunities & promos, and find useful information to guide them through their property acquisition journey.

## My Role

As the frontend lead in a team of four developers, my responsibilities included:

- Setup project architecture
- Automate developers' workflow using Gulp/webpack
- Establish development guidelines
- Analyze requirements & make time estimates
- Setup foundation for core project modules
- Ensure code meets standards through code reviews
- Train junior developers
- Liaise with backend to establish API contracts
- Liaise with UX designer to discuss feasibility
- Participate in client meetings & development

## Key Challenges

- Fast-paced client demands with changing requirements and design
- Integration of external service providers (e.g., 3D guided tours)
- Migration from legacy codebase with unrealistic client expectations
- Tight deadlines
- Cross-platform and cross-device considerations
- Onboarding and training newcomers to the project
